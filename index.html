Lending Club Loan Classification Prediction

Using the Lending Club dataset, I wanted to create a model to better predict whether a loan would be fully paid or charged off. I did preliminary cleaning and feature engineering in SQL, then explored, visualized, and created a predictive model using Python (Jupyter Notebook). The results showed that one of the engineered features (Zip-code, from address) greatly increased the predictive accuracy (~89% vs ~80% baseline). Both a neural network using Tensorflow/Keras and XGBoostClassifier were used, and performed almost exactly the same, so I kept the XGBoost model as the final due to interpretability.

Superstore Sales Forecasting

Using the Superstore Sales data, which is sales from a store containing categories Office Supplies, Technology, and Furniture. I noticed that the average sales per order varied a lot with the three categories, so decided to focus on forecasting only the Furniture category. Digging into this, there were four subcategories, one of which (Furnishings) had a very low average sale per order vs. others, so I split this into two datasets from here, "Furnishings" and "Non-Furnishings". I then explored the data further and use Prophet to forecast individually on the Furnishings and Non-Furnishings data. Despite highly irregular data with too much noise, the model was able to have the mean absolute percentage error on the two for daily predictions to be 89% and 54%, respectively.

Jupyter Notebook

Abalone Age Predictor

This is a Jupyter Notebook for analyzing and creating a regularized linear regression model to predict the age of an abalone using Python. I made use of many of the standard Data Science libraries, including Pandas, Numpy, Matplotlib/Seaborn, Sci-Kit Learn. The regression model performed very well on dataset with less than 14 rings (RMSE ~1.5), which is the majority of the data. There was an RMSE of ~4.5 for the abalone with at least 14 rings.

BodyFat Prediction

This project was focused on predicting bodyfat percentages based on measurements. It is very hard to get an accurate prediction with the naked eye. Cheap methods (using calipers) are very unreliable and accurate methods are very expensive. By perform extensive feature selection and model selection, an ElasticNet model yielded the best and simplest results, which had a mean absolute error of ~3.5%.

Jupyter Notebook
