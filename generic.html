Adventure Works Sales

This project was done using the database AdventureWorks2019 provided by Microsoft. I used SQL Server to create a complex query involving joins, window functions, user created functions, and stored procedures to get a final view, then analyzed the sales data to see what drove an increase in sales over time. As discovered, the promotion done on the new touring bikes worked quite well.

COVID-19 Cases, Deaths, and Vaccinations

I wanted to better understand and visualize how the main data being tracked on COVID would change over time. I found some data by web-scraping and searching for datasets, aggregated created views in SQL Server, then used them in Tableau to visualize cases, deaths, and vaccinations changing over time and how they are related.
